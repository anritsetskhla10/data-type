JavaScript Data Types

Primitive Types

  • String

    Represents textual data, enclosed within single (' '), double (" "), or backtick (` `) quotes.

    "Hello world!"
    String info
  • Number

    Represents numeric data, including integers and floating-point numbers.

    407
    Number info
  • Boolean

    Represents a logical entity with two values: true or false.

    true / false
    Boolean info
  • Undefined

    Represents the absence of a value. Typically, a variable that has not been initialized.

    Undefined

  • Null

    Represents the intentional absence of any object value.

    Null

    Unique and immutable data type used as unique identifiers.

    Symbol('desc')

    Represents integers with arbitrary precision.
